# CrossPoint AirBook Tools

Web flasher and build pipeline for the [CrossPoint Reader + AirBook](https://github.com/Yoddikko/crosspoint-reader) firmware fork.

Adapted from [crosspoint-reader/crosspoint-tools](https://github.com/crosspoint-reader/crosspoint-tools). Serves firmware built from [`Yoddikko/crosspoint-reader`](https://github.com/Yoddikko/crosspoint-reader) instead of the upstream repo, and surfaces upstream sync status on the landing page.

## Related

- [Airbook-for-CrossPoint](https://github.com/Yoddikko/Airbook-for-CrossPoint) — iOS companion app
- [crosspoint-reader](https://github.com/Yoddikko/crosspoint-reader) — firmware source served by the flasher
