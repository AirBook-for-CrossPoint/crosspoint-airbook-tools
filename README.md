# CrossPoint AirBook Tools

Web flasher and build pipeline for the [CrossPoint Reader + AirBook](https://github.com/AirBook-for-CrossPoint/crosspoint-reader-with-aribook) firmware fork.

Adapted from [crosspoint-reader/crosspoint-tools](https://github.com/crosspoint-reader/crosspoint-tools). Serves firmware built from [`AirBook-for-CrossPoint/crosspoint-reader-with-aribook`](https://github.com/AirBook-for-CrossPoint/crosspoint-reader-with-aribook) instead of the upstream repo, and surfaces upstream sync status on the landing page.

Live site: **<https://airbook-for-crosspoint.github.io/crosspoint-airbook-tools/>**
